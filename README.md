# MA3206 Assignment 4

## Algorithms Implemented

1. **Greedy by Food Cost**  
   Selects actions based on the minimum food cost.

2. **Greedy by Food Frequency**  
   Prioritizes foods that appear most frequently.

3. **A\* Search (Optimal Cost)**  
   Uses heuristic-based search to find the minimum cost solution.
